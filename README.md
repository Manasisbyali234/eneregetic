# Energethics Engineering Solutions Website

A professional industrial company website built with React.js and Tailwind CSS.

## Features

- **Responsive Design**: Fully responsive across all devices
- **Modern UI**: Clean and professional design with orange accent color
- **Sticky Navigation**: Fixed navbar with dropdown menu
- **Smooth Animations**: Hover effects and transitions
- **Contact Form**: Functional contact form with validation
- **Component-Based**: Reusable React components

## Components

- **Navbar**: Sticky navigation with dropdown menu and mobile hamburger menu
- **Hero Section**: Large banner with company tagline
- **About Section**: Vision, Mission, and Values
- **Products Section**: Product cards with hover effects
- **Industries Section**: Grid layout of industries served
- **Contact Section**: Contact form and company information
- **Footer**: Three-column footer with links and contact details

## Installation

1. Navigate to the project directory:
```bash
cd energethics-website
```

2. Install dependencies (if not already installed):
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
```

## Technologies Used
